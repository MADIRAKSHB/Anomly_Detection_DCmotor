My project Real-Time Fault Detection in DC Motor using Anomaly Detection focuses on monitoring the health condition of a DC motor and identifying faults at an early stage. In this system, I used the STM32 Nucleo-F411RE microcontroller as the main processing unit because it provides high processing speed, built-in ADC channels, and good support for real-time embedded applications. Sensors such as an accelerometer, current sensor, and temperature sensor are connected to the controller to continuously collect important motor parameters like vibration, current, and temperature. The accelerometer is mainly used to detect abnormal vibration caused by mechanical problems such as bearing faults, imbalance, or shaft misalignment. The collected sensor data is processed by the STM32 controller and compared with the normal operating pattern of the motor. If the real-time values deviate significantly from the normal range, the system identifies it as an anomaly or fault condition. Once an anomaly is detected, the system generates an alert so that maintenance can be performed before the motor fails completely. This method helps in predictive maintenance, reduces unexpected downtime, and improves the efficiency and reliability of the motor system.
